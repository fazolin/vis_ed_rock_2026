# ED ROCK — Guia Visual 2026

Guia visual de referência para o show solo de **Ed Rock** (Racionais MC's), produzido pelo **Ateliê Digital Analógico** — [ada.art.br](https://ada.art.br).

## O que é

Documento único em HTML que reúne o sistema visual completo do show: paleta de cores, conceitos por música, loops de LED com prompts de geração de imagem (frame estático e animação).

13 músicas · 3 blocos narrativos · 52 loops de LED

## Estrutura

| Bloco | Músicas | Cor | Tema |
|-------|---------|-----|------|
| 01 | M01–M04 | #E05C1A | Cultura · Rua · Estilo |
| 02 | M05–M07 | #C9A84C | Origem · Identidade |
| 03 | M08–M13 | #C0392B | Consciência · Luta · Superação |

## Como usar

Abrir `index.html` diretamente no browser — sem servidor, sem dependências.

Os prompts de geração estão protegidos por senha. Solicite acesso à ADA.

## Estrutura de arquivos

```
index.html              — documento principal
style.css               — sistema visual
script.js               — interações (lightbox, unlock, copy)
imagens/
  m01/ … m13/           — frames gerados por música
    loop[n]-frame.png   — frame estático
    loop[n]-anim.mp4    — referência de animação (quando existir)
```
